# Small PyGame game

You are playing as crystal.
Goal is get the coin fast as you can
## Controls:
 * Move: arrows
 * Mute sounds: `M`
 * Restart: `R`
 * Delete save: Hold `X`
 * Pause: `ESC` or `SPACE`

## How to run:
 * Download the repo
 * Install PyGame using `python -m pip install pygame`
 * Run `python main.py` or run `main.py` using python
